Por meio do curso de Node.js da plataforma Alura cursos e, com o estudo de Node.js, 
express e Mongo DB utilizando mongoose e subindo o servidor através do nodemon,
criei uma biblioteca de livros, para fazer a busca e cadastro de livros e autores.

Nesse projeto só foi criado o back-and, então para testar a aplicação é necessário a utilização
do Postman.
